# Purple Board

Purple Board is a Learning Management System (LMS) developed as a university project.  
It provides a GUI-based platform with role-based access for **Admin**, **Instructor**, and **Student** users.

## Features
- Role-based authentication (Admin / Instructor / Student)
- Course and class management
- Assignments and quizzes management
- Student grades tracking
- Relational database integration
- GUI built using Java Swing

## Technologies Used
- Java (Swing)
- SQL (Relational Database)
- Object-Oriented Programming (OOP)

## Database Design
The system is designed using ERD and implemented as a relational database with proper primary and foreign key relationships.
