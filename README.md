# Purple Board

Purple Board is a **Learning Management System (LMS)** developed as a university project.  
The system provides a **GUI-based platform** with **role-based access** for **Admin**, **Instructor**, and **Student** users, supported by a relational database design.

---

## Features
- Role-based authentication (Admin / Instructor / Student)
- Course and class management
- Assignments and quizzes management
- Student grades tracking
- Search and linking functionality
- Relational database integration
- GUI built using Java Swing

---

## Technologies Used
- Java (Swing)
- SQL (Relational Database)
- Object-Oriented Programming (OOP)

---

## Database Design
The system database is designed using **ERD / Logical Schema** and implemented as a relational database with proper **primary and foreign key relationships**.

> Note: The database contains **demo data only** for testing purposes.

## How to Run
1. Open the project in a Java IDE (IntelliJ IDEA or Eclipse).
2. Make sure **Java** is installed on your system.
3. Import the database using the provided SQL file.
4. Run `App.java` to start the application.
